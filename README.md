# Análisis Exploratorio y Geoespacial de Internet Fijo en Perú

¡Bienvenido/a a este repositorio! Este proyecto realiza un análisis detallado y espacial de las conexiones de internet fijo en el territorio peruano, utilizando datos abiertos del sector de telecomunicaciones (OSIPTEL / Ministerio de Transportes).

## 🚀 Objetivo del Proyecto
El propósito es identificar las brechas digitales, la distribución de la infraestructura de red y el comportamiento del mercado de conectividad en las diferentes regiones del país.

## 📁 Estructura del Repositorio
* **DATOS/**: Contiene los datasets limpios y el diccionario de variables.
* **notebooks/**: Scripts y cuadernos de Google Colab con el paso a paso del análisis.
* **outputs/**: Mapas interactivos y animaciones de la evolución de las conexiones (`Mapa.mp4`).

## 🛠️ Tecnologías Utilizadas
* **Python** (Pandas, GeoPandas, Folium/Plotly)
* **Google Colab**
